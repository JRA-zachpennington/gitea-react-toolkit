# Tree Component

The Tree component accepts a tree (an array of tree objects),
or a url to fetch a tree from.

It is a recursive component that renders more trees when it encouters
an object in the tree of type tree.

## Examples

### Example 1

A manually provided tree listing.

<Playground>
<Paper>
  <Tree
    selected
    tree={[
      {
        "path": ".github",
        "type": "tree",
        "tree": [
          {
            "path": "config",

          },
        ]
      },
      {
        "path": "LICENSE.md",
        "type": "blob",
      }
    ]}
  />
</Paper>
</Playground>
